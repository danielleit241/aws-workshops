# Đề xuất dự án

Phần **Proposal** dùng để đặt bản đề xuất dự án / ý tưởng.

### Nội dung gợi ý

- **Tóm tắt điều hành (Executive Summary)** — Tóm tắt dự án, mục tiêu, phạm vi.
- **Tuyên bố vấn đề (Problem Statement)** — Vấn đề hiện tại và giải pháp đề xuất.
- **Kiến trúc giải pháp (Solution Architecture)** — Kiến trúc kỹ thuật, tech stack.
- **Kế hoạch triển khai (Implementation Plan)** — Lộ trình, milestone, ước lượng chi phí (nếu có).

Bạn có thể đính kèm file PDF: đặt file vào `static/files/Proposal/` và thêm link tải trong nội dung (vd: `<a href="/files/Proposal/Your_Proposal.pdf" download>Proposal PDF</a>`).

Ảnh minh họa đặt trong `static/images/Proposal/`.
